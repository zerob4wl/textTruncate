# Text Truncate

> Truncating Styled Text to Fit a Container Using JQuery


## Getting Started


In your web page:

```html
<script type="text/javascript">
  $(document).ready(function() {
  $('#element').textTruncate({
        marginText :  "...",
        percentOfParent : "50",
        parent : '#element-parent'
    })
  });
</script>
```
